# What is red teaming?

Red team can mean different things, but in the context of this wiki it refers to the methodology of attacking an organization in a way a modern adversary would. It involves compromising an organization using techniques apart from usual digital exploitation like Social Engineering.

To be a "red teamer" you need know about certain basic topics.
